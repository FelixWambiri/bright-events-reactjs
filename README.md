## Installation
```
$ git clone git@github.com:younggeeks/bright-events-reactjs.git bright-events
$ cd bright-events
```
Install Dependencies:
```
$ yarn 
```
Then run the app:
```
$ yarn start
```
To see your application, access this url in your browser: http://localhost:3000
